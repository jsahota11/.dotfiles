# Legend for integrating .dotfiles

---

## nvim

Just link the 'nvim' folder into ~/.config

## kitty

Just link the contents into ~/.config/kitty

## bash

Just link the contents into ~

## zsh

Just link the contents into ~

## oh-my-zsh

Inside is the 'custom' folder; link the whole 'custom' folder
into ~/.oh-my-zsh
